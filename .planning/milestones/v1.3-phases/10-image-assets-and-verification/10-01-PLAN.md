# Plan 10-01: Image Assets And Zip Export

**Phase:** 10 Image Assets And Verification
**Status:** Completed

## Goal

Extract embedded images during conversion, write stable Markdown image paths,
and add zip export for Markdown plus assets.

## Tasks

- Add generated asset state to the converter.
- Use Mammoth `convertImage` to capture image bytes locally.
- Name assets as `image-N.ext` and return `./assets/image-N.ext` as Markdown
  image sources.
- Add asset summary UI.
- Add zip download containing `document.md` and `assets/`.

## Verification

- Image documents produce Markdown references under `./assets/`.
- Zip export includes Markdown and extracted asset files.
- No upload, R2, Worker, or persistent storage path is introduced.
