# Plan 07-02: Preview Fidelity And Catalog Polish

## Objective

Make the catalog trustworthy enough for repeated daily use by stabilizing
previews and removing misleading data states.

## Tasks

- Audit the imported data and filter out families that cannot present a usable
  preview.
- Prefer generated static preview images while keeping a safer fallback preview
  path for gaps.
- Simplify the page framing so the prompt content stays primary.

## Must Haves

- Visible cards do not ship obviously broken preview states.
- Template switching does not produce unexplained empty prompt content.
- The page remains readable on desktop and mobile.
