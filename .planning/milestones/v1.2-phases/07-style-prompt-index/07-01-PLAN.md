# Plan 07-01: Local Prompt Catalog

## Objective

Build the first shippable `Style Prompt Index` experience inside the tools
branch.

## Tasks

- Add the `/tools/style-prompt` route and browser component.
- Prepare a local style prompt dataset that can be searched and filtered in the
  frontend.
- Present the catalog as a card gallery with a single modal detail flow.

## Must Haves

- The route feels like a searchable prompt index.
- Cards expose preview, naming, and browsing value without requiring navigation.
- Users can copy Chinese or English prompt text directly from the modal.
