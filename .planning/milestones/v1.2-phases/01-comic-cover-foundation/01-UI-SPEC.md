# Phase 1 UI Spec: Comic Cover Foundation

## Visual Contract

- First viewport is a full-screen comic cover.
- `MAYS UNIVERSE` is the only visible hero text.
- Title treatment uses heavy black stroke, stacked color shadows, skew/rotation, and high weight.
- Background uses print texture, burst composition, and action-line energy.
- A small animated cue suggests continuation without readable instruction text.

## Quality Bars

- The page must not look like a normal gradient landing page.
- Title must fit on desktop and mobile without overlapping nearby UI.
- Motion must be subtle and disabled under `prefers-reduced-motion`.
- The cover should not introduce navigation behavior; it is visual setup for the panel directory below.
