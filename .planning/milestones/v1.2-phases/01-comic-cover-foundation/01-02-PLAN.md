# Plan 01-02: Comic Texture, Motion, And Responsive Cover

## Objective

Implement the comic cover visual language and make it responsive.

## Tasks

- Add heavy outline, halftone, speed-line, and burst styling in `styles/portal.module.css`.
- Add a subtle non-text motion cue.
- Add responsive breakpoints so the title fits on narrow screens.
- Add reduced-motion handling.

## Must Haves

- Cover reads as old-school American comic design.
- Title remains dominant on desktop and mobile.
- Motion is not required for comprehension.
