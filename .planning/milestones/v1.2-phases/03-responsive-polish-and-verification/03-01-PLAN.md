# Plan 03-01: Responsive And Accessibility Polish

## Objective

Make the redesigned portal reliable across viewport sizes and basic navigation modes.

## Tasks

- Add mobile breakpoints for cover title, panel sizing, and panel typography.
- Add accessible labels for portal links and sealed buttons.
- Add reduced-motion behavior.
- Keep not-found page styles usable.

## Must Haves

- No obvious text overflow in the core layout.
- Interactive elements have accessible names.
- Motion can be reduced.
