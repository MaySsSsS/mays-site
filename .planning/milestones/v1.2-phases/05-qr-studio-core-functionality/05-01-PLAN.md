# Plan 05-01: QR Studio Live Composer

## Objective

Build the interactive QR editor and preview surface.

## Tasks

- Add a `/tools/qr` route for the tool.
- Create a client component that manages content, size, margin, and color
  controls.
- Render the QR preview locally in the browser as settings change.

## Must Haves

- Accept both plain text and URLs.
- Show immediate on-page output.
- Keep the implementation frontend-only.
